# Django Production Deployment Guide

The book is built with MkDocs and is ready to publish on Read the Docs.

- Read the source at [`docs/index.md`](docs/index.md).
- Preview locally with `mkdocs serve`.
- Follow [`docs/read-the-docs-setup.md`](docs/read-the-docs-setup.md) to publish it.

Install the documentation dependencies with:

```bash
python -m pip install -r requirements-docs.txt
```
